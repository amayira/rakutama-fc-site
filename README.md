# 楽珠そろばん教室 FC加盟店募集サイト

株式会社ワンパレード（畠中卓人代表）の楽珠そろばん教室フランチャイズ加盟店募集サイト。
公開想定URL: fc.rakutama-soroban.com（サブドメイン要確定・DNSは畠中さん側管理）

## 構成

```
public/            ← デプロイ対象（静的HTML・ビルド不要）
├ index.html       トップ：紹介経由・指名検索の受け皿。属性分岐カードで2LPへ
├ owner/           経営者・フリーランス向けLP ★Meta広告の着地先
├ teacher/         ホームティーチャー向けLP  ★Meta広告の着地先
├ voice/           加盟店の実績（練馬・枚方・藤沢）
├ contact/         説明会申込フォーム（属性ラジオつき共通版）
├ thanks/          サンクスページ（?from=owner|teacher でLead CV計測）
├ css/style.css    共通スタイル（デザイントークンは archive/ のプロトタイプ継承）
├ js/site.js       フォーム送信＋UTM引き継ぎ
└ img/             ロゴ・校舎写真（rakutama-lp-tokyo から流用）
archive/           2026-06版Reactプロトタイプ（数値が旧条件のため未使用・参照用）
```

## 設計方針（2026-07-14 有山さん確定）

- **広告LPとHPは分離しない**。/owner・/teacher は単体で説得が完結するフルLPとし、
  Meta広告（属性別ターゲティング）はここへ直行。トップからの分岐導線も張る
- /owner・/teacher のヘッダーは常時スリム（ロゴ＋CTAのみ）。出し分けロジックは持たない
- 掲載数値の一次資料は `01_そろばん教室事業/楽珠そろばん教室FC説明資料.pdf`（2026-07-14版）。
  **数値を変える時は必ずPDF最新版と突合すること**
- 「紹介者様限定」の建付けは広告経由を枠外とすることで畠中さんと合意済み。
  フォームの紹介者欄は任意で残す
